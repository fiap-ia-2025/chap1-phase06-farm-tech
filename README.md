# 🌾 FarmTech Solutions - Projeto de IA para Agricultura

## 📋 Sobre o Projeto

O **FarmTech Solutions** é um projeto de Inteligência Artificial desenvolvido para a **Fase 6 do curso de Inteligência Artificial da FIAP**. O projeto visa a implementação de arquiteturas YOLO e CNN para detectar e segregar objetos a fim de criar um sistema de visão computacional eficaz.

### 🎯 Objetivos

- **Dataset**: seleção de imagens separadas em treinamento, validação e teste.
- **Labels**: criação de labels nas imagens para identificação dos objetos
- **Arquiteturas**: YOLO e CNN
- **Validação**: avaliar performance dos modelos
